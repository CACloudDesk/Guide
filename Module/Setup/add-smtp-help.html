Add SMTP

How SMTP works in CA Cloud Desk: choose your email provider (Gmail, Yahoo, Rediff, or Outlook) and follow the steps to create an app password (where required) and configure SMTP in CA CloudDesk so notifications and documents can be sent from your account.

Overview

SMTP (Simple Mail Transfer Protocol) is used by CA Cloud Desk to send emails—for example notifications and documents. You configure one or more email accounts in Settings → SMTP Configuration. For Gmail, Yahoo, and Outlook you must use an App Password (not your normal login password). For Rediff Mail you use your normal email password. Port 587 is used for all providers below.

Path & screens

CA CloudDesk → Settings → SMTP Configuration

From the dashboard, open Settings (e.g. via your profile or main menu), then open SMTP Configuration. Use the SMTP Configuration tab to enter your email and server details; use View SMTP to see saved configurations.

Dashboard — navigate to Settings (e.g. via profile or menu) to reach SMTP Configuration.
Settings — click SMTP Configuration to open the form.
SMTP Configuration form — enter Email, Password, Host, Port; optionally set Default, then click Save.

Option 1: Gmail

Gmail Option 1

1

Create App Password in Google

  • Log in to your Google Account.
  • Click Manage your Google Account.
  • Go to Security.
  • Enable Two-Step Verification (if not already on).
  • Search for App Passwords.
  • Generate a new App Password.
  • Copy the password (you will use it in CA CloudDesk).
2

Configure Gmail SMTP in CA CloudDesk

Go to CA CloudDesk → Settings → SMTP Configuration. Select Gmail (if the option is available) and enter:

FieldValue
Email AddressYour Gmail ID
SMTP Hostsmtp.gmail.com
Port587
UsernameYour Gmail ID
PasswordApp Password (the one you generated)

Optionally check Default, then click Save.

✅ Gmail SMTP is now configured.

Option 2: Yahoo Mail

Yahoo Mail Option 2

1

Create App Password in Yahoo

  • Log in to your Yahoo Account.
  • Go to Account Security.
  • Enable Two-Step Verification.
  • Click Generate App Password.
  • Select Mail app.
  • Copy the password.
2

Configure Yahoo SMTP in CA CloudDesk

Go to CA CloudDesk → Settings → SMTP Configuration. Select Yahoo and enter:

FieldValue
Email AddressYour Yahoo ID
SMTP Hostsmtp.mail.yahoo.com
Port587
UsernameYour Yahoo ID
PasswordApp Password

Click Save.

✅ Yahoo SMTP is now configured.

Option 3: Rediff Mail

Rediff Mail Option 3

1

Use Your Rediff Email Password

App password is not required for Rediff. Use your normal Rediff email password.

2

Configure Rediff SMTP in CA CloudDesk

Go to CA CloudDesk → Settings → SMTP Configuration. Select Rediff and enter:

FieldValue
Email AddressYour Rediff ID
SMTP Hostsmtp.rediffmail.com
Port587
UsernameYour Rediff ID
PasswordYour Rediff Email Password

Click Save.

✅ Rediff SMTP is now configured.

Option 4: Outlook

Outlook Option 4

1

Create App Password in Outlook

  • Log in to your Microsoft Account.
  • Go to Security.
  • Enable Two-Step Verification.
  • Click App Passwords.
  • Generate a new App Password.
  • Copy the password.
2

Configure Outlook SMTP in CA CloudDesk

Go to CA CloudDesk → Settings → SMTP Configuration. Select Outlook and enter:

FieldValue
Email AddressYour Outlook ID
SMTP Hostsmtp.office365.com
Port587
UsernameYour Outlook ID
PasswordApp Password

Click Save.

✅ Outlook SMTP is now configured.
